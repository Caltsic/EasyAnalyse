### 3.3 方向模型只保留 input / output
现行模型将 terminal 的 `direction` 收敛为两种持久化类型：`input` 与 `output`。

`direction` 允许：
- `input`
- `output`

约定：
- I2C `SCL`、`SDA` 这类连接也只保留 `input / output`，按当前文档里更重要的阅读方向建模
- 电阻、电容、电感、磁珠、晶振、开关等两端器件也必须显式写成 `input / output`
- 电源输入、地端通常写 `input`
- 稳压输出、被驱动的信号源通常写 `output`

布局默认侧边直接由 `direction` 推导：`input -> left`，`output -> right`。如果发布展示需要更稳定的版式，再显式补 `side` 与 `order`。
